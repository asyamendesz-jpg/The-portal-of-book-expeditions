# The-portal-of-book-expeditions
«Портал книжных экспедиций»— интерактивный проект для детей 7–12 лет. Каждая книга становится увлекательным путешествием с аудиоэпизодами, заданиями, викторинами, комиксами и творческими мастерскими. Читай, исследуй, открывай новые миры и становись настоящим исследователем книг!
Книги, аудиоэпизоды, рабочие тетради, викторины, карточки персонажей, творческие задания и книжные путешествия, которые превращают чтение в увлекательное приключение.
