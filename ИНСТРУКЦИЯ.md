# Портал книжных экспедиций — инструкция

## Как открыть сайт

1. Откройте папку проекта на компьютере.
2. Запустите **локальный сервер** (без него PDF могут не скачиваться):

```powershell
cd "путь\к\папке\внеклассное чтение"
py -m http.server 8080
```

3. В браузере перейдите по адресу: **http://localhost:8080**

> Не открывайте `index.html` двойным щелчком (`file://`) — ссылки на PDF работают только через сервер.

---

## Маршрут экспедиции для ребёнка

1. **Главная** — ознакомиться с маршрутом, нажать «Начать экспедицию».
2. **Книжные путешествия** — выбрать книгу.
3. **Герои книг** — выбрать книгу и познакомиться с персонажами.
4. **Экспедиционные комплекты** — скачать PDF-материалы.
5. **Карточки персонажей** — собрать коллекцию героев.
6. **Викторины и загадки** — пройти испытание после чтения.

## Где хранятся данные

| Массив | Файл | Что добавлять |
|--------|------|---------------|
| `books[]` | `js/data/books.js` | Новые книги-экспедиции |
| `heroes[]` | `js/data/heroes.js` | Герои книг |
| `cards[]` | `js/data/cards.js` | Карточки персонажей |
| `workbooks[]` (PDF) | `js/data/workbooks.js` | PDF-комплекты |
| `quizzes[]` | `js/data/quizzes.js` | Викторины |
| `passports[]` | `js/data/passports.js` | Паспорта |
| `comics[]` | `js/data/comics.js` | Комиксы читателей |
| `pages[]`, `banners` | `js/data/pages.js` | Меню и баннеры |

HTML менять **не нужно** — достаточно дописать объект в нужный массив.

## Как добавить новую книгу

Откройте `js/data/books.js` и добавьте объект:

```javascript
{
  id: 'new-book-id',
  title: 'Название книги',
  author: 'Автор',
  route: 'Тематика маршрута',
  color: '#4a7cb8',
  cover: 'images/covers/new-book.png'
}
```

Положите обложку в `images/covers/`.

---

## Как добавить героя

1. Добавьте портрет в `images/heroes/`.
2. Откройте `js/data/heroes.js`:

```javascript
{
  id: 'hero-id',
  name: 'Имя героя',
  role: 'Роль',
  description: 'Краткое описание.',
  bookId: 'alice-journey',
  image: 'images/heroes/hero-id.png',
  color: '#e87461'
}
```

3. При необходимости добавьте карточку в `js/data/cards.js` с тем же `bookId`.

---

## Как добавить PDF-комплект

1. Положите файл в папку `pdf/`.
2. Добавьте запись в `js/data/workbooks.js`:

```javascript
{
  id: 'kit-new',
  title: 'Название комплекта',
  description: 'Описание.',
  bookId: 'alice-journey',
  file: 'pdf/имя-файла.pdf'
}
```

---

## Как заменить баннер страницы

1. Положите изображение в `images/mockups/`.
2. Обновите путь в `js/data/pages.js` → объект `banners`:

```javascript
heroes: {
  image: 'images/mockups/geroi-knig.png',
  alt: 'Описание для доступности',
  title: 'Герои книг',
  intro: 'Текст под баннером.'
}
```

## CSS — где что искать

| Файл | Содержание |
|------|------------|
| `css/layout.css` | Каркас, header, footer, отступы |
| `css/navigation.css` | Боковое меню |
| `css/components.css` | Баннеры, экспедиция, блоки |
| `css/cards.css` | Все типы карточек |
| `css/buttons.css` | Кнопки |
| `css/responsive.css` | Адаптив для телефона и планшета |
| `css/theme.css` | Анимации и hover-эффекты |

Главный файл `css/style.css` подключает все модули через `@import`.

## Советы

- Используйте **короткие латинские имена** для файлов изображений (`alice.png`, не пробелы).
- После изменения данных обновите страницу в браузере (Ctrl+F5).
- Новые викторины пока ведут на `href: '#'` — замените на ссылку, когда викторина будет готова.
